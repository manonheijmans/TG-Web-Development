Opdracht 1

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

::first line: kan gebruikt worden om de eerste letter van een paragraph anders te stijlen.

::after: creeert een pseudo-element dat het laatste child is van het geselecteerde element. Het wordt vaak gebruikt in combinatie met de content property.

:selection: hiermee kun je stijl toevoegen aan een deel van een document dat is gehighlight door een gebruiker.

::file-selector-button: representeert de button van een input van type="file".
