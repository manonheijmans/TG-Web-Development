Opracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

De selector wijst naar het HTML element dat je wilt stylen.
Het declaration blok bevat een of meerdere declarations gescheiden door puntkomma's.
Elke declaration bevat de naam van een CSS property en een waarde, gescheiden door dubbele punten.
Meerdere CSS declarations worden gescheiden door puntkomma's. Declaration blokken worden ingesloten door accolades.
