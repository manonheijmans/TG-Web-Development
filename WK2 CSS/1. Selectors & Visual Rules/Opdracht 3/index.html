Er zijn verschillende selectors, bijvoorbeeld; type selectors, descendant selectors, class selectors. Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.
  • Type selector: Met een type selector selecteer je direct een element. Door p { } te gebruiken, kun je alle paragraph elementen tegelijk stijlen.
  • Descendant selector: De descendant selector matcht alle elements die descendants zijn van het genoemde element. div p { } selecteert alle paragraph elementen binnen div elementen.
  • Class selector: De class selector selecteert elementen die als attribute een specifieke class hebben meegekregen. Een p met als class attribute "intro" kun je stijlen met .intro { }
