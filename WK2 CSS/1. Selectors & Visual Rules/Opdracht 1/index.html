Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  • Inline: Inline CSS betekent dat je de styling direct in je HTML element toevoegt
  • Internal stylesheet: De styling wordt toegevoegd in de head van je HTML document
  • External stylesheet: Styling gebeurt door middel van een extern document, waarnaar je linkt in de head van je HTML document
