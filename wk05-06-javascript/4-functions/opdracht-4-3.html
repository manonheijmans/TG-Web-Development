Eerder heb je kennis gemaakt met het String object en de built-in String Methods. Nu gaan we kennismaken met nog zo een - het Math object. Het Math object heeft namelijk ook methods die bij functies erg veel gebruikt worden:

a. Maak een functie randomNumber() en verbindt die aan een button. Zorg dat de functie Math.random() in de console logt. Wat merk je op als je deze meerdere keren gebruikt? 

    <button onclick="randomNumber()">Random number button</button>

    function randomNumber(){
        console.log(Math.random());
    }

Meerdere keren loggen levert allemaal cijfers tussen 0 en 1 op.

Math.random() geeft dus een getal 0 >= x < 1 terug - het kan wel 0 zijn, maar nooit 1. Als je die laat afronden met Math.round(), zal het vaker afronden naar beneden dan naar boven.
Als je een willekeurig getal wil tonen, moet je Math.floor() gebruiken op Math.random(). Als je dit zo doet, krijg je altijd 0 terug.

b. Verander je functie nu dat hij een parameter x gebruikt, en zorg dat hij het volgende in je console logt: Math.floor(Math.random() * x). Test nu je functie met verschillende waardes voor x. Valt iets je op?

    <button onclick="randomNumber2(variabeleX)">Random number button 2</button>

    function randomNumber2(x){
        console.log(Math.floor(Math.random() * x));
    }

    let variabeleX = 10

Het cijfer dat je aan de parameter x toekent, geeft nu de boven en ondergrens van de random uitkomsten aan. Bij x = 10, wordt er altijd 0 t/m 9 gelogd.

c. Er zijn nog 2 belangrijke Math methods die handig zijn om te kennen. Onderzoek wat Math.min() en Math.max() doen. Kan je van beide al een voorbeeld geven?

The static function Math.min() returns the lowest-valued number passed into it, or NaN if any parameter isn't a number and can't be converted into one.
The Math.max() function returns the largest of the zero or more numbers given as input parameters, or NaN if any parameter isn't a number and can't be converted into one.

Math.max example

 
    function mathMaxExample(x, y, z){
    return Math.max(x, y, z);
    }

    let resultMax = mathMaxExample(1, 8, 3)

    console.log(resultMax)

Math.min example

    function mathMinExample(x, y, z,){
        return Math.min(x, y, z);
    }

    let resultMin = mathMinExample(2,10,6)

    console.log(resultMin)

In het math.Max voorbeeld wordt het hoogste getal gelogd. In het math.Min voorbeeld het laagste getal.
