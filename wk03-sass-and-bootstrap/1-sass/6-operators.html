Opdracht 1

.image {height: 400px * 2; width: auto;}
wordt:
.image {height: 800px; width: auto;}

Opdracht 2

.image { height: 400px + 100; width: auto;}
wordt:
.image {height: 500px; width: auto;}
Ik had verwacht dat deze niet zou werken, omdat 100 niet als pixels wordt gedefinieerd. Schijnbaar neemt ie dit dus wel over.

Opdracht 3

image {height: 400px - 300px; width: auto;}
wordt:
.image {height: 100px; width: auto;}

Opdracht 4

.image {height: 200px + 100px - 400px; width: auto;}
wordt:
.image {height: -100px; width: auto;}
Ook hier had ik een error verwacht. Een hoogte kan niet negatief zijn. Ik krijg echter geen foutmelding.

Opdracht 5

.image {height: 200px * 3; width: auto;}
wordt:
.image {height: 600px; width: auto;}

Opdracht 6

.image {height: 750px * 1px; width: auto;}
Je kunt niet vermenigvuldigen met 1px, dit zal een foutmelding opleveren. De juiste manier is:
.image {height: 750px * 1; width: auto;}

Opdracht 7

.image {height: 30px / 5px; width: 24px/ 4;}
Hier gaan twee dingen mis. Enerzijds wordt er gedeeld door px, wat niet kan. Anderzijds moet met haakjes ( ) aangegeven worden dat / hier gebruikt wordt om te delen. Zo zou de code eruit moeten zien:
.image {height: (300px / 2); width: (24px / 4);}

Opdracht 8

.image {height: (30px / 5px); width: (24px/ 4);}
De haakjes ( ) gaan hier wel goed. Het delen door pixels niet. De code zou er zo uit moeten zien:
.image {height: (30px / 5); width: (24px/ 4);}
